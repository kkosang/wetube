#YouTube Clone Coding
Vanilla & NodeJS를 이용한 Youtube Clone

/ -> Home
/join -> Join
/login -> Login
/search -> Search

/user/:id -> See user
/user/logout -> Log Out
/user/edit -> Edit MY Profile
/user/delete -> Delete MY Profile

/videos/:id -> See Video
/videos/:id/edit -> Edit Video
/videos/:id/delete -> Delete Video
/videos/upload -> Upload Video
